### ✨ Features
*   🌐 **Bilingual Interface:** Toggle between English and Thai languages instantly.
*   ⌨️ **Quick Submit:** Press `Enter` on your keyboard inside the input field to generate instantly.
*   📐 **Aspect Ratio Selector:** Choose from 1:1, 3:4, 4:3, 16:9, 9:16, 4:6 (Etsy Print), and 3:5.
*   🖼️ **Multi-Image Grid:** Generate 1 to 8 images at once with different seeds, displayed in a beautiful Civitai-style grid.
*   📝 **SEO Listing Data:** Automatically writes SEO Title, engaging Description, and 13 keywords (Tags) ready for Etsy.
*   🎨 **Civitai-Style UI:** Premium dark cyberpunk theme with purple/blue neon glow effects.

### 🚀 Getting Started

#### Prerequisites
*   Windows OS
*   Python 3.10+ installed ([Download Python](https://www.python.org/downloads/))
*   Free Gemini API Key ([Get API Key Here](https://aistudio.google.com/app/apikey))

#### Installation & Run
1.  **Download the project:** Click the green **Code** button above > **Download ZIP** > Extract the files.
2.  **Add your API Key:**
    *   Create a folder named `API Key` in the project root directory.
    *   Create a file named `API_KEY.txt` inside that folder.
    *   Paste your Gemini API Key in the file (just the raw key, single line) and save.
3.  **Run the App:** Double-click `start.bat`. Your browser will open the app automatically.

### 🚀 วิธีการติดตั้งและใช้งาน

#### สิ่งที่ต้องเตรียม
*   ระบบปฏิบัติการ Windows
*   ติดตั้ง Python เวอร์ชั่น 3.10 ขึ้นไป ([ดาวน์โหลดที่นี่](https://www.python.org/downloads/))
*   Gemini API Key ฟรีจาก Google AI Studio ([กดขอที่นี่](https://aistudio.google.com/app/apikey))

#### วิธีใช้งาน
1.  **ดาวน์โหลดโปรเจกต์:** กดปุ่ม **Code** สีเขียวด้านบน > **Download ZIP** > แตกไฟล์ ZIP
2.  **ใส่รหัสผ่าน API Key:**
    *   สร้างโฟลเดอร์ชื่อ `API Key` ไว้ในโฟลเดอร์หลักของโปรเจกต์
    *   สร้างไฟล์ชื่อ `API_KEY.txt` ในโฟลเดอร์ดังกล่าว
    *   นำ API Key ไปวางไว้ในไฟล์นั้น (บรรทัดเดียวเพียวๆ) แล้วกดบันทึก
3.  **เริ่มใช้งาน:** ดับเบิ้ลคลิกไฟล์ `start.bat` หน้าเว็บแอปพลิเคชันจะเปิดขึ้นในเบราว์เซอร์ของคุณโดยอัตโนมัติ
